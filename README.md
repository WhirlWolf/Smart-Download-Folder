# Smart-Download-Folder
Automatically sort files in the download folder based on the file type.
</div>

---

## Prerequisites
- [Tasker](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm)

## Setup Instructions
1. [Install](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm) and setup tasker.
2. [Import](https://taskernet.com/shares/?user=AS35m8m8L9YzBV3qbzaAAqHiSYXYBbD3QfZ7hr0hRK4ojOFTCrjWh2CScbjMw4NaudRi1zKKzq85&id=Project%3AWuzzApp) project and follow setup instructions.
3. Ensure the profile is enabled.

## Features
- Create sorting rules
- Quick actions: Open, rename, move, delete, locate, extract, share, undo, etc
- Check properties
- Copy name, copy path, copy md5
- Compare md5
- Move, extract to specific directory
- Image preview
- Move tasker files
- Creates folders only when required
- Auto rename if file with the same name already exists
- Exportable as a standalone app

## Tips
- Make sure tasker is allowed to run in background
- Increase maximum tasks queued in tasker > preferences > action
