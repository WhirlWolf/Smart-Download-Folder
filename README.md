# Smart-Download-Folder
Automatically sort files in Download folder based on the file type.

## Prerequisites
- [Tasker](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm)

## Setup Instructions
1. Install and setup tasker
2. [Import](https://taskernet.com/shares/?user=AS35m8m8L9YzBV3qbzaAAqHiSYXYBbD3QfZ7hr0hRK4ojOFTCrjWh2CScbjMw4NaudRi1zKKzq85&id=Project%3AWuzzApp) and follow setup instructions

## Features
- Rename if file already exists
- Create sorting rules
- Quick actions: Open, rename, move, delete, locate, extract, share, undo, etc
- Check properties
- Copy name, copy path, copy md5
- Compare md5
- Move, extract to specific directory

## Tips
- Make sure tasker is allowed to run in background
- Increase maximum tasks queued in tasker > preferences > action
